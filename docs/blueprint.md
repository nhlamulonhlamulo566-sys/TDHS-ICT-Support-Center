# **App Name**: GPHealth ICT Support Center

## Core Features:

- Help Desk Ticketing: Allows help desk staff to log and categorize incoming IT support calls.
- Technician Call Assignment: Assign incoming calls to specific technicians based on expertise and availability.
- Technician Interface: Technicians can view, update, and resolve assigned tickets. Also includes an escalate feature.
- Senior Technician Escalation: Senior technicians can manage escalated tickets and re-assign to supervisors if necessary.
- Supervisor Oversight: Supervisors monitor escalated tickets, manage resource allocation, and ensure timely resolutions.
- Resolved Ticket Archive: A repository of resolved tickets, searchable by various parameters (technician, date, issue type, etc.).
- Admin User Management: Admin tool to manage Helpdesk, Technician, Senior Technician, and Supervisor user accounts and permissions.

## Style Guidelines:

- Primary color: Deep blue (#2E3192) to convey trust and professionalism.
- Background color: Light gray (#F0F2F5) for a clean, uncluttered interface.
- Accent color: Teal (#008080) for interactive elements and call-to-action buttons.
- Body and headline font: 'Inter', a sans-serif font providing a modern and neutral look.
- Use a consistent set of professional, line-based icons to represent call status, severity, and category.
- A clean, structured layout with clear hierarchy and intuitive navigation.
- Subtle animations (e.g., transitions, loading indicators) to enhance user experience without being distracting.
